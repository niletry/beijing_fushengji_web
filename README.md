# 北京浮生记 Web版

## 🎮 游戏介绍

这是经典DOS游戏《北京浮生记》的现代Web复刻版本。在90年代的北京，通过倒卖商品、应对随机事件，积累财富，成为商业巨子！

## ✨ 特色

- **🎨 现代化设计** - 赛博朋克风格遇上90年代怀旧美学
- **⚡ 流畅动画** - 精心设计的过渡效果和交互反馈
- **📱 响应式布局** - 完美适配各种设备
- **💾 本地存储** - 支持保存/读取游戏和排行榜
- **🎲 随机事件** - 丰富的随机事件系统
- **📊 市场动态** - 动态价格系统和市场波动

## 🎯 游戏玩法

### 游戏目标
在限定天数内积累尽可能多的财富

### 核心机制
- **倒卖商品** - 在不同地点买入卖出，赚取差价
- **地点旅行** - 8个不同地点，每个都有独特的价格特点
- **随机事件** - 好运与厄运并存，考验你的应变能力
- **资金管理** - 存款获利息，合理分配现金和存款
- **健康管理** - 保持健康，否则游戏提前结束
- **背包升级** - 租更大的房子增加背包容量

### 地点列表
- 🌉 **天桥** - 平民市场，生活用品便宜
- 🏬 **东单** - 商业中心
- 🚇 **西直门** - 交通枢纽
- 🏰 **王府井** - 高档商业区
- 💻 **中关村** - 电子产品市场
- 🏢 **国贸CBD** - 国际商务区
- 🌊 **后海** - 酒类市场
- 🍸 **三里屯** - 时尚潮流区

### 商品类型
- 🥮 烧饼
- 🍶 二锅头
- 🚲 自行车
- 📺 电视机
- 📱 大哥大
- 💻 电脑
- 🍾 茅台
- 🔥 Zippo打火机

## 🎮 游戏提示

1. **寻找价差** - 不同地点同一商品价格差异很大，多走访观察
2. **及时存款** - 银行每天有0.5%利息，长期收益可观
3. **健康第一** - 健康值归零游戏结束，该治疗就治疗
4. **扩大容量** - 背包容量限制了进货量，适时升级租房
5. **关注新闻** - 市场事件会大幅影响商品价格，抓住机会

## 🚀 启动游戏

### 方法1: 直接打开
```bash
# 使用Python的简单HTTP服务器
python3 -m http.server 8000

# 或使用Node.js的http-server
npx http-server -p 8000
```

### 方法2: 使用VS Code Live Server
1. 在VS Code中打开项目
2. 安装Live Server扩展
3. 右键点击`index.html`选择"Open with Live Server"

然后在浏览器中访问 `http://localhost:8000`

## 🔧 自定义

你可以轻松修改游戏参数：
- 在`game.js`中修改`GOODS`数组改变商品
- 修改`LOCATIONS`数组添加新地点
- 调整`RANDOM_EVENTS`数组自定义事件
- 在CSS中修改配色方案
